# Boundary Lab

> 邊界條件不對稱 × 場耦合 × 量子真空 — 一個業餘研究者的精密測量開源實驗倉庫，研究歷史上反覆出現但未被主流物理接受的無推進劑推進與重力耦合現象（俗稱「反重力」）。

這是一個業餘興趣項目，不是新創公司。一個普通上班族在下班與週末時段，與 Claude Opus 長期協作，對主流物理學早已停止認真看的一類聲稱做誠實的精密測量。

我們大概不會發現反重力。但若我們做得夠仔細，至少能很有信心地說*「沒發現」* — 在這個領域，這就已經是貢獻了。

---

## 這個倉庫是什麼

一份公開、可複製、開放授權的實驗紀錄，追蹤以下研究主題：

- Townsend Brown 非對稱電容器效應（Biefeld–Brown effect）
- Podkletnov 旋轉超導盤重力屏蔽聲稱
- Buhler / Exodus Propulsion 的靜電壓力推進聲稱
- McCulloch 的量子化慣性（Quantized Inertia, QI）理論
- Fetisov 等人 2022 年常溫 YBCO + RF 重量變化複製實驗

我們的工作假說（**假說，不是結論**）是：上述現象若為真，可能共享同一個底層機制 — 「邊界條件不對稱 + 場耦合 + 量子真空」。

## 這個倉庫**不是**什麼

為了避免誤導讀者、投資者或新聞媒體，請明確了解：

- ❌ 這**不是**一個聲稱已經發現反重力的項目。
- ❌ 這**不是**一個募資、商業推進、或代幣項目。
- ❌ 這**不是**陰謀論、自由能源、永動機、或反主流科學的平台。
- ❌ 這**不是**一個保證會有成果的研究計畫 — 大多數結果預期將為零或不顯著。
- ✅ 這**是**一個公開記錄一系列嚴格設計的測量實驗的工作日誌。
- ✅ 這**是**一個歡迎獨立複製、批判、與否證的協作平台。

## 這個倉庫的位置

這是一個**業餘研究者**的長期實驗紀錄。維護者是一名上班族，自費維持本研究 — 沒有外部資助、沒有商業計畫、不期待獲利。

開源不是行銷策略，是因為一個人玩沒意思 — 我們希望找到志同道合的人**一起玩、一起研究**。

歡迎你加入，如果你符合以下三條：

1. **你享受精密測量** — 對抗熱漂移、特性化雜訊本底、看到一張漂亮的 PSD 圖會有點開心。
2. **你願意分享負面結果** — *「我試了 X，沒看到任何效應」*就是我們要的數據。
3. **你不期待靠這個倉庫致富** — 沒有 exit。回報是工作本身、和一起做的人。

如果這三條聽起來對，你來對地方了。

## 為什麼開源

過去七十年的相關研究（Brown、Podkletnov、Grebennikov、Hayasaka、Ning Li、Woodward 等）有一個結構性問題：**資料封閉、複製困難、職業生涯風險高**。多數研究者最終流落於小型私營機構或失去主流資源。我們相信這個領域的進展被這個結構性問題嚴重拖延。

公開所有實驗協議、原始資料、分析腳本、失敗紀錄，是我們認為這個領域唯一可能突破的方式。即使我們的所有實驗都得到零結果，這些零結果本身也是有科學價值的貢獻 — Tajmar 等人 2021 年對 EmDrive 的精密證偽論文是這個領域近年最重要的工作之一。

## 科學紀律（核心承諾）

本倉庫所有實驗與發布遵守以下不可協商的紀律：

1. **無誇大**：任何標題、摘要、commit 訊息、社群媒體貼文中，不使用「突破」「成功」「驗證」等用語，除非已通過獨立第三方複製。
2. **零結果同等發表**：失敗、無效應、或不顯著的實驗結果，與正面結果以同等格式、同等優先級發布。
3. **原始資料永不刪改**：`data/raw/` 中的所有檔案為 append-only；任何資料處理皆於 `analysis/` 中重新產生，原始檔不修改。
4. **方法先於結果**：每個實驗的 SOP（標準操作程序）必須在實驗執行前提交至 `protocols/` 並打 git tag。事後修改 SOP 不視為原 SOP。
5. **聲稱必有引用**：任何科學聲稱必須附上文獻引用或原始資料連結。AI（Claude）產生的合成性敘述需明確標註並接受同等審查。
6. **誤差優先**：任何測量結果必須伴隨誤差分析、本底雜訊、與系統性誤差的討論。沒有誤差的數字不是科學數據。
7. **獨立可複製**：實驗的硬體 BOM（Bill of Materials）、接線圖、校準紀錄必須完整公開於 `hardware/`。任何其他研究者應能依文件複製。
8. **AI 協作透明化**：本倉庫的工作流程包含 Claude Opus 作為長期協作者。AI 在實驗設計、文獻整理、資料分析中的具體貢獻於 `AI_CONTRIBUTIONS.md` 持續記錄。

違反上述任一條的內容應被回滾、修正或公開撤回。

## 安全聲明（極度重要）

本倉庫涉及的實驗包含可致命的風險。在執行任何實驗前，操作者必須：

- 高壓電源（>1 kV）：使用接地手套、絕緣墊、單手操作規則、雙人在場、緊急斷電開關。
- 液氮：僅於通風空間操作，配戴護目鏡與低溫手套；密閉空間有窒息風險。
- 強磁鐵（N42 以上）：注意夾傷與飛擊；遠離心律調節器與磁性儲存裝置。
- 真空艙：使用合格容器；不可使用一般玻璃；內爆風險。
- 電容器：高壓電容器即使斷電後仍可能殘留致命電壓；放電前必先以接地棒短路。

本倉庫的內容**僅供研究參考**。任何複製本倉庫實驗者應自行負責安全。維護者不對任何因複製本倉庫內容而造成的傷害承擔法律責任。

## 如何加入我們

歡迎以下形式的參與：

- **獨立複製**本倉庫已發布的實驗，**特別是失敗的複製** — 那才是這個領域真正缺的。
- **批判性審查** SOP、誤差分析、或結論。
- **文獻補充與翻譯**（特別是俄文與日文文獻）。
- **硬體設計改進** — 即便只是指出*「你接線圖畫錯了」*也歡迎。

提交 Issue 或 Pull Request 時，請遵守上述「科學紀律」中的引用與誤差規範。我們不是公司 — 願意聊聊勝過走流程。
